为什么选择 Java 版？

无限的模组 (Mods)

Java 版拥有最庞大的模组社区，从工业科技到魔法冒险，彻底改变游戏玩法。

自由的服务器

支持搭建私人服务器，拥有 Hypixel 等全球最大的小游戏服务器网络。

自定义皮肤与材质

完全开放的资源包系统，让你可以替换游戏中的每一个像素和音效。

老牌稳定

Minecraft Forge

历史最悠久，支持大型科技/魔法模组最多。兼容性最好，但启动速度稍慢。
